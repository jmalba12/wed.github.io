Mi glosario personal de Sistemas Operativos

Por:Manuel Alba



A continuación encontrarás las definiciones de las principales palabras y expresiones relacionadas con la asignatura de Sistemas Operativos


INCOTEC

La INCOTEC es una organización sin fines de lucro de Colombia fundada en 1963, encargada de supervisar el cumplimiento de las normas nacionales e internacionales.



Ethernet

Ethernet es un estándar de red que conecta dispositivos entre sí mediante cableado de par trenzado. Es el protocolo de red más común en las LAN.



WEB

La Web es una red global de computadoras que permite el intercambio de información mediante páginas web conectadas entre sí mediante hipervínculos.



DHCP

DHCP es un protocolo de red que asigna automáticamente direcciones IP y otros parámetros de configuración de red a los dispositivos conectados a la red. Esto simplifica la administración de la red.



LAN

LAN es una red informática que conecta los ordenadores de un área local. Se utiliza para compartir recursos y proporcionar acceso a Internet.



MAN

MAN es una red informática que conecta los ordenadores de un área metropolitana.



IP

IP es un conjunto de reglas que definen cómo se envían y reciben los datos por Internet. Cada dispositivo conectado a Internet tiene una dirección IP única, que se utiliza para dirigir los paquetes de datos a su destino correcto. IP es esencial para el funcionamiento de Internet.



HTTPS

HTTPS es una versión segura del protocolo HTTP que utiliza encriptación para proteger los datos transmitidos entre un navegador web y un sitio web. HTTPS es ideal para transmitir datos confidenciales, como números de tarjetas de crédito y contraseñas.



TIA

TIA es una organización que representa a las empresas de telecomunicaciones en Colombia. Trabaja para promover el desarrollo del sector y garantizar que los colombianos tengan acceso a servicios de telecomunicaciones de alta calidad.



EIA

AIE es una organización que representa a las empresas de electrónica en Colombia. Trabaja para promover el desarrollo del sector y garantizar que los colombianos tengan acceso a productos y servicios electrónicos de alta calidad.



Sistema Operativo

El sistema operativo es un software que gestiona los recursos de hardware de una computadora y proporciona servicios a los programas de aplicación. Controla tareas como la administración de archivos, la administración de memoria y la administración de procesos.



Núcleo del Sistema

El núcleo del sistema es la parte central de un sistema operativo que se encarga de la administración de procesos, la administración de memoria y la administración de dispositivos. Es esencial para el funcionamiento del sistema operativo.



Interfaz de Usuario

La interfaz de usuario es la parte del sistema operativo que permite a los usuarios interactuar con la computadora. Puede ser una interfaz gráfica de usuario (GUI) o una interfaz de línea de comandos (CLI).



Proceso

Un proceso es un programa en ejecución en una computadora. El sistema operativo gestiona múltiples procesos y les asigna recursos como CPU y memoria.



Memoria Virtual

La memoria virtual es una técnica que permite a un sistema operativo utilizar espacio en disco como si fuera memoria RAM. Esto amplía la capacidad de la memoria y permite ejecutar programas más grandes de lo que cabrían en la memoria física.



Administrador de Tareas

El administrador de tareas es una herramienta en el sistema operativo que permite al usuario ver y controlar los procesos en ejecución, la utilización de la CPU y otros recursos del sistema.



Archivo de Paginación

El archivo de paginación es un archivo en disco utilizado por el sistema operativo para almacenar datos de memoria que no caben en la RAM física. Ayuda a mantener un equilibrio entre la cantidad de memoria utilizada y el rendimiento del sistema.



Kernel de Sistema

El kernel de sistema es la parte interna del sistema operativo que interactúa directamente con el hardware de la computadora. Es responsable de realizar tareas críticas como la administración de dispositivos y la gestión de interrupciones.



Administración de Archivos

La administración de archivos es una función del sistema operativo que controla la creación, lectura, escritura y eliminación de archivos en un sistema de almacenamiento. Organiza y supervisa el acceso a los archivos y directorios.
