Mi glosario personal de Redes

Por: Manuel Alba



A continuación encontrarás las definiciones de las principales palabras y expresiones relacionadas con la asignatura de Redes


Router

Un router es un dispositivo que dirige el tráfico de red entre diferentes redes o subredes. Se utiliza para enrutar paquetes de datos y conectar dispositivos en una red.



Protocolo IP

El Protocolo de Internet (IP) es un conjunto de reglas que gobierna la forma en que los datos se transmiten a través de una red. Es esencial para la comunicación en Internet.



Firewall

Un firewall es un dispositivo o software que protege una red al controlar y filtrar el tráfico de red entrante y saliente. Ayuda a prevenir amenazas de seguridad.



Switch

Un switch es un dispositivo de red que conecta dispositivos dentro de una red local (LAN) y dirige el tráfico de datos a los destinatarios correctos. Es más inteligente que un hub.



Servidor DNS

Un servidor DNS (Sistema de Nombres de Dominio) es responsable de traducir nombres de dominio legibles por humanos en direcciones IP que las computadoras puedan entender. Es fundamental para la navegación web.



Subred

Una subred es una subdivisión de una red IP más grande. Permite una gestión más eficiente de las direcciones IP y la segmentación de la red para mejorar la seguridad y el rendimiento.



Enrutamiento

El enrutamiento es el proceso de determinar la mejor ruta para enviar datos a través de una red. Los routers toman decisiones de enrutamiento en función de la información de la tabla de enrutamiento.



Proxy

Un proxy es un intermediario que actúa como un servidor intermedio entre un cliente y un servidor final. Se utiliza para mejorar la seguridad y el rendimiento de la red.



Wi-Fi

Wi-Fi es una tecnología de red que permite la conexión inalámbrica de dispositivos a Internet y entre sí. Es comúnmente utilizado en redes domésticas y empresariales.



Cable Ethernet

Un cable Ethernet es un cable de red utilizado para conectar dispositivos mediante conexiones con cable. Es común en redes LAN y proporciona conexiones de alta velocidad.



Protocolo TCP/IP

El Protocolo de Control de Transmisión/Protocolo de Internet (TCP/IP) es el conjunto de protocolos que sustenta Internet. Proporciona comunicaciones confiables y se utiliza para la mayoría de las comunicaciones en línea.



Red Inalámbrica

Una red inalámbrica es una red de comunicación que se basa en la transmisión de datos sin cables. Es conveniente para dispositivos móviles y entornos donde no se pueden utilizar cables.



Topología de Red

La topología de red se refiere a la disposición física o lógica de los dispositivos y conexiones en una red. Puede ser en estrella, bus, anillo u otras configuraciones.



Servidor Web

Un servidor web es un software o hardware que almacena y entrega páginas web a través de Internet. Es esencial para la disponibilidad de sitios web en línea.
